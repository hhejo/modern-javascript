# 2.8 기본 연산자와 수학

### 단항, 이항, 피연산자

## 수학

`+`, `-`, `*`, `/`, `%`, `**`

## 이항 연산자 `+`와 문자열 연결

문자열 연결 가능

문자열과 숫자도 연결 가능

```javascript
alert("1" + 2); // "12"
alert(2 + "1"); // "21"
alert(2 + 2 + "1"); // "41"
alert(6 - "2"); // 4
alert("6" / "2"); // 3
```

## 단항 연산자 `+`와 숫자형으로의 변환

피연산자가 숫자가 아닌 경우 숫자형으로 변환

```javascript
alert(+true); // 1
alert(+""); // 0
alert(+"2" + +"3"); // 5
```
